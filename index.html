  • CLI project process using git:
  • 1: In CLI, navigate to the repo project stored on your device
  • 2: Check status for current repo branch and any pending commits with: git status
  • 3: If you aren't in your desired branch, navigate with: git checkout branchName
  • 4: If you need to create a new branch use: git checkout -b newBranchName
  • 5: Before working on new content, to avoid issues with version-control, make sure you're operating from most recent version of the main branch with: git pull origin mainBranchName
  • 6: Check status once more as a final version/pending check

  • When wanting to push to main branch using CLI:
  • 1: Update to most recent isntance of main branch with: git pull origin main
  • 2: Stage new content from you branch with: git add -A
  • 3: Commit this staged content with: git commit -m "add a simple description of feature"
  • 4: Optional - update to recent of main branch one last time for good measure ^.^
  • 5: Create a Pull Request to the main branch with: git push origin newBranchName
  • 6: Follow up with team/self in whatever version control software being used
